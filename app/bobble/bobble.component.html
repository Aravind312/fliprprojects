Booble

Bobble Keyboard is an input method that allows users to personalise conversations. Users can convert text into visual elements like stickers, GIFs, emojis, custom fonts etc. Stickers and GIFs can be personalised with users' own faces. It uses AI to suggest relevant content using factors like text, emotions, location, language, relationship etc. Leading smartphone brands in India have chosen it as the default keyboard. It supports English and 21 other Indian languages. It has almost 5% share among Indic keyboards- second only to Google.
